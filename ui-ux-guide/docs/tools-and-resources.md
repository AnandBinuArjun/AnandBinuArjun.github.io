# Tools and Resources for UI/UX Developers

A comprehensive list of tools, resources, and learning materials for UI/UX developers at all levels.

## Design Tools

### Industry Standard Tools

#### Figma
- **Platform**: Web, Windows, macOS
- **Features**: Real-time collaboration, prototyping, design systems
- **Best For**: Team collaboration, web and mobile design
- **Learning Resources**: 
  - [Figma Learn](https://www.figma.com/resources/learn-design/)
  - [Figma YouTube Channel](https://www.youtube.com/c/Figmadesign)

#### Sketch
- **Platform**: macOS only
- **Features**: Vector editing, symbols, plugins
- **Best For**: macOS users, icon design
- **Learning Resources**:
  - [Sketch Tutorials](https://www.sketch.com/learn/)
  - [Sketch YouTube Channel](https://www.youtube.com/user/sketchapp)

#### Adobe XD
- **Platform**: Windows, macOS
- **Features**: Voice prototyping, auto-animate, collaboration
- **Best For**: Adobe Creative Suite users
- **Learning Resources**:
  - [Adobe XD Learn](https://helpx.adobe.com/support/xd.html)
  - [Adobe XD YouTube Channel](https://www.youtube.com/user/AdobeXD)

### Prototyping Tools

#### InVision
- **Platform**: Web, macOS, Windows
- **Features**: Presentation mode, collaboration, design system management
- **Best For**: Presenting designs to stakeholders

#### Marvel
- **Platform**: Web, macOS, Windows
- **Features**: Simple prototyping, user testing
- **Best For**: Rapid prototyping and testing

#### Principle
- **Platform**: macOS
- **Features**: Advanced animation, layer-based animation
- **Best For**: Micro-interaction design

## Development Tools

### Code Editors

#### Visual Studio Code
- **Platform**: Windows, macOS, Linux
- **Features**: Extensions, Git integration, debugging
- **Extensions for UI/UX**:
  - Live Server
  - Prettier
  - Color Highlight
  - Auto Rename Tag

#### Sublime Text
- **Platform**: Windows, macOS, Linux
- **Features**: Speed, multiple cursors, command palette

### Browser Developer Tools

#### Chrome DevTools
- **Features**: Element inspection, performance analysis, device simulation
- **Learning Resources**:
  - [Chrome DevTools Documentation](https://developers.google.com/web/tools/chrome-devtools)

#### Firefox Developer Tools
- **Features**: CSS grid inspection, accessibility tools

## User Research Tools

### Analytics

#### Google Analytics
- **Features**: User behavior tracking, conversion analysis
- **Best For**: Understanding user behavior on existing products

#### Hotjar
- **Features**: Heatmaps, session recordings, surveys
- **Best For**: Visualizing user interactions

### Testing Platforms

#### UserTesting
- **Features**: Remote user testing, video feedback
- **Best For**: Getting qualitative feedback from real users

#### Maze
- **Features**: Prototype testing, analytics
- **Best For**: Testing interactive prototypes
