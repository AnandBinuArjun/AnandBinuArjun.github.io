# Digital Forensics Analyst Guide

A comprehensive guide to becoming a Digital Forensics Analyst, from basics to advanced techniques.

## Description

This repository contains a complete guide to digital forensics, covering everything from fundamental concepts to advanced techniques. The site is designed with a cybersecurity theme featuring a dark background and neon green accents.

## Features

- Responsive design that works on all devices
- Interactive navigation with smooth scrolling
- Terminal-style animations and effects
- Comprehensive content covering all aspects of digital forensics
- Cybersecurity-themed styling with dark mode and neon accents

## Deployment to GitHub Pages

1. Create a new repository on GitHub
2. Push the contents of this folder to your repository
3. Go to Settings > Pages
4. Select the main branch as the source
5. Click Save
6. Your site will be available at `https://[username].github.io/[repository-name]/`

## Customization

You can customize the colors by modifying the CSS variables in `styles.css`:
- `--primary-dark`: Main background color
- `--accent-green`: Primary accent color
- `--text-light`: Light text color
- `--card-bg`: Card background color

## Technologies Used

- HTML5
- CSS3 (Flexbox, Custom Properties)
- JavaScript (ES6+)
- Responsive Design
- CSS Animations
